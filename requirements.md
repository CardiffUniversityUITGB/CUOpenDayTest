1. Functional Requirements
Display the Open Day schedule in a clear, interactive format (cards, list, or grid).
Allow users to filter events by:
Faculty
Time of day
Event type (talk, tour, workshop)
Highlight featured or important events.
Clicking an event shows details (description, location, speaker).

2. Non-Functional Requirements
Fully responsive layout (desktop, tablet, mobile).
Fast loading and smooth UI interactions.
Works on all modern browsers.
Accessible: WCAG 2.1 compliance (keyboard navigation, screen readers, contrast).

3. UI / UX Requirements
Apply Cardiff University branding (colors, fonts, logo).
Clear navigation bar or header.
Event grid or cards with hover effects and spacing.
Optional: “Featured Events” section at the top.

4. Technical Requirements
Use the existing CUOpenDayTest repository as the base.
Modern JavaScript (ES6+) or React (if applicable).
CSS framework optional (Tailwind, Bootstrap, or custom CSS).
Event data should be rendered dynamically from the existing feed.
